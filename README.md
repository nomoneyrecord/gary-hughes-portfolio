# Gary Hughes Portfolio

A full-stack portfolio website built with Angular and Spring Boot to showcase projects, writing, certifications, and professional contact information.

## Overview

This project is designed to present my work as a backend-leaning software engineer with experience in Java, Spring Boot, Angular, Android, SQL, and full-stack academic application development. A chatbot will be added in the future.

The site includes:
- A polished landing page
- A portfolio page with project cards and modal-based project details
- A blog page with dedicated article pages
- A certifications page with official learning links
- A contact page with direct professional links

## Tech Stack

### Frontend
- Angular
- TypeScript
- HTML
- CSS

### Backend
- Java
- Spring Boot
- Maven

## Current Features

- Multi-page Angular site with routing
- Shared application shell with sticky navigation
- Home page with featured project, writing, and certification snapshots
- Portfolio page with reusable project modal pattern
- Dedicated blog post pages
- Certifications page with official external links
- Contact page for professional outreach
- Organized frontend/backend monorepo structure
